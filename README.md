# Install-isdas-in-R-LinuxMint
I had difficulty installing the isdas package into my R library on my laptop with LinuxMint OS. The reason for this was because I was missing a bunch of dependencies for the package. The info for installing these dependencies is available online, but I figured this will save people the googling. This assumes you already have R installed and working on your device, if not, go do that first! Code is for LinuxMint, I'm not sure if it'll work for Ubuntu or other versions of Linux, but if you're someone who uses Ubuntu I'm sure you have the technical know-how to figure out how to alter the code to suit your distro yourself ;)
